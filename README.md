# Basic Tetris

A minimal, dependency-free Tetris implementation using HTML5 Canvas + vanilla JS.

How to run:

1. Start a static server in this folder
   - Python: `python3 -m http.server 8000`
2. Open `http://localhost:8000` in your browser

Controls:
- Left/Right: Move
- Up: Rotate
- Down: Soft drop
- Space: Hard drop
- P: Pause
- R: Restart
